## 📖 About This Repository

This repository contains the source code for my personal portfolio website. It's designed to be a clean, professional, and fast-loading single-page application that highlights my skills, professional experience, and achievements in the field of **Data Engineering, Cloud Infrastructure, and Data Warehousing**.

Built with **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**, this portfolio emphasizes performance, responsiveness, and a sleek dark-themed UI without the overhead of complex build tools.

---

## ✨ Features

- 📱 **Fully Responsive Design**: Optimized for seamless viewing on mobile, tablet, and desktop devices.
- 🌙 **Modern Dark Theme**: A sleek, eye-friendly dark color scheme tailored for developer portfolios.
- ⚡ **Smooth Scrolling Navigation**: Sticky navbar with smooth scroll effects to navigate between sections effortlessly.
- 🎨 **Interactive UI Elements**: Subtle fade-in animations on scroll, hover effects, and gradient accents for an engaging user experience.
- 🚀 **Zero Build Process**: Utilizes Tailwind CSS via CDN for instant styling without needing Node.js or bundlers.
- ♿ **Accessible & Semantic**: Built with semantic HTML5 tags for better SEO and accessibility.

## 🚀 Local Development

Getting this project running locally is straightforward. You don't need any complex dependencies installed.

### Option 1: Using VS Code Live Server (Recommended)

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/agusmahari/agusmahari.github.io.git
    cd agusmahari.github.io
    ```

2.  **Open in VS Code**:
    ```bash
    code .
    ```

3.  **Run Live Server**:
    *   Install the "Live Server" extension if you haven't already.
    *   Right-click on `index.html` and select **"Open with Live Server"**.

### Option 2: Direct Browser Open

Simply navigate to the folder where you cloned the repo and double-click `index.html` to open it in your default browser. Note that some animations might behave differently without a local server due to browser security policies on local files.
