Personal portfolio website showcasing my journey as a Data Engineer specializing in Cloud, Data Warehousing, and ETL/ELT pipelines.

## 🚀 Features
- **Responsive Design:** Works on mobile, tablet, and desktop.
- **Dark Theme:** Modern dark UI built with Tailwind CSS.
- **Performance:** Vanilla JavaScript, no heavy frameworks.
- **Animations:** Subtle scroll reveal effects.

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Font Awesome Icons
